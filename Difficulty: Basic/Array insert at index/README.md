<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>

<pre><span style="font-size: 18px;"><strong>Input: </strong>arr[] = [1, 2, 3, 4, 5], index = 2, val = 90
<strong>Output: </strong>[1, 2, 90, 3, 4, 5]<strong>
